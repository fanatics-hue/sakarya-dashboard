Dashboard Live — Guida di Setup

SAKARYA GAS FIELD DEVELOPMENT PROJECT · IQS S.R.L.

Fase 1 — Google Sheets (fonte dati)
1
Carica il file Excel su Google Sheets
Vai su sheets.google.com → clicca su File → Importa → Carica → seleziona il file Sakarya_Dashboard_DataSource.xlsx. Seleziona "Sostituisci foglio di calcolo".
2
Pubblica il foglio come CSV
Nel Google Sheet vai su File → Condividi → Pubblica sul Web. Seleziona il foglio SUMMARY, formato Valori separati da virgola (.csv) → clicca Pubblica.
3
Copia l'URL generato
Copia l'URL che appare (esempio: https://docs.google.com/spreadsheets/.../pub?gid=0&single=true&output=csv). Tienilo da parte.
Fase 2 — Collegare il Dashboard
4
Apri il file HTML del dashboard
Apri il file dashboard_sakarya_v2.html con un editor di testo (Notepad++, VS Code, o anche il Blocco Note di Windows).
5
Incolla l'URL Google Sheets
Cerca nel file il testo YOUR_GOOGLE_SHEET_CSV_URL_HERE e sostituiscilo con l'URL copiato al passo 3. Salva il file.
Fase 3 — Pubblicare su GitHub Pages
6
Crea un account GitHub (se non ce l'hai)
Vai su github.com → Sign up → crea account gratuito.
7
Crea un nuovo repository
Clicca su New repository → nominalo sakarya-dashboard → impostalo come Public → clicca Create repository.
8
Carica il file HTML
Nel repository clicca Add file → Upload files → carica dashboard_sakarya_v2.html → rinominalo in index.html → clicca Commit changes.
9
Abilita GitHub Pages
Vai su Settings → Pages → in "Source" seleziona main branch → cartella / (root) → clicca Save. Dopo 1-2 minuti il sito sarà live.
10
Condividi il link
Il tuo dashboard sarà accessibile all'indirizzo https://[tuo-username].github.io/sakarya-dashboard/. Condividi questo link con i colleghi.
Fase 4 — Aggiornamento Dati
★
Come aggiornare i dati
Apri il Google Sheet → modifica i valori nel foglio SUMMARY (es. Incoming Plates, Accepted, ecc.) → il dashboard si aggiorna automaticamente entro 5 minuti per tutti gli utenti, senza toccare il file HTML.
★
Come aggiungere nuovi difetti
Aggiungi righe nei fogli DEFECTS_WELD o DEFECTS_FINAL. Pubblica anche questi fogli come CSV separati e inserisci gli URL nel dashboard.
✅ Risultato finale: Un link fisso condivisibile con chiunque, che mostra sempre i dati aggiornati in tempo reale direttamente dal Google Sheet — senza server, senza costi.
⚠️ Nota privacy: Il Google Sheet rimane privato. Solo il CSV pubblicato è accessibile — e contiene solo i dati che vuoi mostrare.
